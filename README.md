## Travelle - Tourism Website

### Features

Users will able to:

- Login and register to the website.
- After login the name will appear on header
- see the upcoming Tour
- Book a tour
- Manage bookings
- Add Tours
- animated 404 page.
- Private route that can be accessed only after login

### Built with

- React Component
- React Route
- Node
- MongoDb
- Firebase
- Bootstrap
- HTML
- CSS
